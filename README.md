# Common-ML-techniques-from-Scratch
Implementation of Logistic Regression, Neural Networks, Decision Trees and Random Forests from scratch and comparision by using libraries, with hyperparameter tuning and performance evaluation.


## Models Implemented
- Logistic Regression (from scratch)
- Logistic Regression (library)
- Neural Network (from scratch)
- Neural Network (TensorFlow)
- Decision Tree
- Random Forest

## Datasets Used
- IMDb Movies Dataset
- GTZAN Music Genre Dataset

## Evaluation Metrics
- Accuracy
- F1 Score

## Key Concepts Covered
- Forward propagation
- Backpropagation
- Gradient descent
- Hyperparameter tuning
- Model comparison

## How to Run
Open the notebook and run all cells.

Dataset paths follow Kaggle format:
`/kaggle/input/...`
